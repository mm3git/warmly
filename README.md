# 💝 Warmly — Family Wellness App

A simple, real-time family app that motivates aging parents to stay active through love and recognition.

## Quick Start

👉 **Live App:** [Open Warmly](./warmly-app-production.html)

## Setup Instructions

1. **First time?** Read [WARMLY_SETUP_GUIDE.md](./WARMLY_SETUP_GUIDE.md)
2. **In a hurry?** Use [WARMLY_QUICK_START.md](./WARMLY_QUICK_START.md) checklist

## What is Warmly?

- Mom in LA logs daily activities (stretch, walk, play games)
- Sisters in Tokyo and Taipei see it in real-time
- Grandkids can send encouragement and compete on leaderboards
- Bilingual (Chinese for Mom, English for family)
- Free to host, runs on Supabase

## Features

✅ Activity logging (Stretch, Walk, Games)  
✅ Real-time family sync  
✅ Daily rotating games  
✅ Leaderboards & streaks  
✅ Bilingual interface (中文 + English)  
✅ Mobile-friendly PWA
