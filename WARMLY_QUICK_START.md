# Warmly — Quick Start Checklist

**Copy & paste this into a task list or notes app. Check off as you go.**

---

## 🚀 Get Live (Next 15 Minutes)

- [ ] Go to https://supabase.com → Sign Up → Create account
- [ ] Create new project named `warmly-family`
- [ ] Go to Settings → API
- [ ] Copy **Project URL** (save temporarily)
- [ ] Copy **API Key (anon)** (save temporarily)
- [ ] Open `warmly-app-production.html` in browser
- [ ] Click "Setup Supabase"
- [ ] Paste URL → Paste API Key → Click "Save"
- [ ] Select **Mark (你)** and enter
- [ ] Go to https://netlify.com → Sign Up
- [ ] Drag `warmly-app-production.html` into Netlify
- [ ] Wait for deploy
- [ ] Copy your Netlify URL (the `https://xxx.netlify.app` link)

---

## 📱 Test it Works (2 Minutes)

- [ ] Open Netlify URL in two browser windows
- [ ] Left window: Log in as **Mom**
- [ ] Right window: Log in as **Mark**
- [ ] As Mom: Click "I stretched today"
- [ ] As Mark: See activity appear in real-time ✅

## 📊 Monitor First Week

- [ ] Day 1-2: Check if Mom logs in, what she clicks
- [ ] Day 3-4: Note interaction patterns (stickers only? or real messages?)
- [ ] Day 5-7: Ask Mom — what motivates her to log in?
- [ ] Gather any technical issues

## 🎯 What You're Testing

- Does Mom actually use the app daily?
- What features get the most engagement?
- Do family messages motivate her more than app notifications?
- Do games encourage repeat usage?

All this real data will inform the next build and your investor pitch.
